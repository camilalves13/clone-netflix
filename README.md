## Clone da Netflix

Clone da primeira página da Netflix, em HTML com slider para ver os filmes disponiveis.

#### Tecnologias
* HTML5
* CSS3
* JAVA SCRIPT
* JQUERY
* ALGUNS CONCEITOS DE FLEX BOX NO CSS.
* PÀGINA RESPONSIVA (Desktop, Mobile and Tablet).
